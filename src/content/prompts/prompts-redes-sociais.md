# 15 Prompts para Redes Sociais com IA

Manter presenca ativa em redes sociais exige volume e consistencia. Estes prompts ajudam voce a criar conteudo para as principais plataformas de forma rapida, mantendo a qualidade e a identidade da sua marca.

## Bio e Perfil

### 1. Bio Otimizada por Plataforma

Para criar bios que convertem visitantes em seguidores.

```
Crie bios otimizadas para os seguintes perfis de [marca/profissional de segmento]:

Informacoes:
- Nome: [nome]
- O que faz: [descreva em 1 frase]
- Publico-alvo: [quem quer atrair]
- Diferencial: [o que te torna unico]
- CTA desejado: [seguir, clicar no link, enviar DM]

Gere bio para:
1. Instagram (150 caracteres, com quebras de linha estrategicas)
2. LinkedIn (220 caracteres, tom profissional)
3. Twitter/X (160 caracteres, tom conciso)
4. TikTok (80 caracteres, tom casual)

Para cada bio:
- Use a primeira linha para dizer O QUE voce faz
- Use a segunda linha para QUEM voce ajuda ou QUAL resultado entrega
- Termine com CTA claro
- Sem jargoes ou frases vagas como "apaixonado por"
```

### 2. Link na Bio (Estrutura da Pagina)

Para organizar o conteudo do link na bio de forma estrategica.

```
Crie a estrutura de uma pagina de "link na bio" para [marca/profissional].

Objetivo principal: [captar leads / vender / redirecionar para conteudo]

Organize os links em ordem de prioridade:
1. [Link principal - ex: produto, mentoria, ebook gratuito]
2. [Conteudo mais recente - ex: ultimo video, artigo]
3. [Prova social - ex: depoimentos, portfolio]
4. [Contato - ex: WhatsApp, email]
5. [Outras redes sociais]

Para cada link, escreva:
- Texto do botao (maximo 30 caracteres, com verbo de acao)
- Descricao curta abaixo do botao (maximo 50 caracteres)

Sugira: cor de fundo, ferramenta recomendada (Linktree, Beacons, etc.) e como testar qual link mais converte.
```

## Instagram - Captions e Legendas

### 3. Lote de Legendas para a Semana

Para produzir conteudo de uma semana inteira de uma vez.

```
Crie 7 legendas de Instagram (1 por dia) para [marca/perfil de nicho].
Publico: [descreva].
Tom de voz: [descreva ou de exemplos].

Distribuicao semanal:
- Segunda: motivacional/mindset (comeco de semana)
- Terca: dica pratica/tutorial (conteudo de valor)
- Quarta: bastidores/dia a dia (humanizar)
- Quinta: conteudo educativo aprofundado
- Sexta: engajamento/enquete/pergunta
- Sabado: entretenimento/tendencia/leveza
- Domingo: reflexao/storytelling pessoal

Para cada legenda:
- Texto: 100-200 palavras
- Gancho na primeira linha (o que aparece antes do "mais")
- CTA no final (comentar, salvar, compartilhar, mandar para alguem)
- 10 hashtags (mix de populares e nicho)
- Sugestao de formato do post (foto, carrossel, reels)
```

### 4. Legendas para Lancamento de Produto

Para criar uma sequencia de posts que gera expectativa e converte.

```
Crie uma sequencia de 5 posts de Instagram para o lancamento de [produto/servico].
Data do lancamento: [data].

Post 1 (7 dias antes): Teaser - gerar curiosidade sem revelar
Post 2 (5 dias antes): Problema - amplificar a dor que o produto resolve
Post 3 (3 dias antes): Revelacao - apresentar o produto
Post 4 (dia do lancamento): Oferta - preco, bonus, link
Post 5 (2 dias depois): Prova social - depoimentos/resultados dos primeiros clientes

Para cada post:
- Legenda completa (150-250 palavras)
- Formato sugerido (carrossel/reels/estatico)
- Briefing visual (o que mostrar na imagem/video)
- Hashtags (10)
- Melhor horario para publicar
```

## Instagram - Stories e Reels

### 5. Sequencia de Stories Interativos

Para criar stories que geram engajamento e respostas.

```
Crie uma sequencia de 8 stories para [marca/perfil] sobre [tema].
Objetivo: [engajamento/vendas/educacao/branding].

Para cada story:
- Tipo: [texto/foto/video/enquete/quiz/slider/pergunta/contagem regressiva]
- Texto na tela (maximo 20 palavras)
- Instrucao visual (o que mostrar no fundo)
- Sticker interativo sugerido
- Duracao sugerida (se video)

Regras:
- Story 1: gancho que prende (pergunta ou afirmacao forte)
- Stories 2-6: conteudo/narrativa
- Story 7: interacao (enquete, quiz ou caixa de perguntas)
- Story 8: CTA (link, DM, ou "arrasta pra cima")

O conjunto deve contar uma mini-historia do inicio ao fim.
```

### 6. Roteiros de Reels por Formato

Para criar reels em diferentes estilos que funcionam no algoritmo.

```
Crie 5 roteiros de Reels de [15/30/60] segundos para [marca/perfil de nicho].
Cada um em formato diferente:

1. Tutorial rapido: "Como [fazer algo] em [X] passos"
   - Texto na tela para cada passo + narracacao

2. Mito vs Realidade: desmontar uma crenca sobre [tema]
   - Split screen ou antes/depois

3. Trending audio: adaptar a tendencia [descreva] para o nicho de [nicho]
   - Roteiro sincronizado com o audio

4. Lista/Ranking: "Top [X] [coisas] que todo [publico] precisa saber"
   - Transicoes rapidas entre itens

5. Storytelling: mini-historia pessoal/da marca sobre [tema]
   - Narrativa em primeira pessoa

Para cada roteiro: gancho (primeiros 2 segundos), roteiro completo, texto de legenda, hashtags e audio sugerido.
```

## Hashtags

### 7. Estrategia de Hashtags

Para criar um sistema de hashtags que funciona de verdade.

```
Crie uma estrategia de hashtags para [perfil/marca de nicho] no Instagram.
Seguidores atuais: [numero].
Publico-alvo: [descreva].
Tipo de conteudo: [educativo/entretenimento/vendas].

Forneca:
1. 10 hashtags de nicho (menos de 100k posts - menor concorrencia)
2. 10 hashtags medias (100k-500k posts)
3. 5 hashtags populares (500k-5M posts)
4. 5 hashtags da marca (proprias, para comunidade)

Para cada hashtag: nome + numero estimado de posts + relevancia para o nicho.

Alem disso:
- 3 conjuntos prontos de 15 hashtags para usar em: post educativo, post de venda, post de engajamento
- Hashtags a evitar (banidas ou muito genericas)
- Dica de posicionamento (na legenda ou no primeiro comentario)
```

## LinkedIn

### 8. Post de LinkedIn com Storytelling

Para criar posts que geram engajamento profissional.

```
Escreva um post de LinkedIn sobre [tema/experiencia/opiniao].
Meu perfil: [cargo, setor, anos de experiencia].

Estrutura que funciona no LinkedIn:
- Gancho (primeira linha visivel): frase impactante ou controversa
- Historia pessoal ou observacao (3-4 paragrafos curtos)
- Insight ou licao aprendida
- Pergunta para engajamento no final

Regras:
- Paragrafos de 1-2 frases (facilitar leitura no mobile)
- Maximo 200 palavras
- Sem hashtags no corpo (3 hashtags no final)
- Sem links no post (prejudica alcance - colocar no comentario)
- Tom: [autoritativo/vulneravel/provocativo/educativo]

Gere 3 versoes com abordagens diferentes.
```

### 9. Artigo Longo para LinkedIn

Para publicar conteudo aprofundado na plataforma.

```
Escreva um artigo de LinkedIn sobre [tema].
Meu perfil: [cargo, expertise].
Publico: [quem quero atingir].

Estrutura:
- Titulo chamativo (maximo 70 caracteres)
- Introducao: por que este tema importa agora (100 palavras)
- 3-5 secoes com subtitulos
- Dados, exemplos ou cases para sustentar cada ponto
- Conclusao com opiniao pessoal e CTA (comentar, seguir, conectar)

Tom: especialista que compartilha conhecimento generosamente. Nao e um artigo academico - e uma conversa entre profissionais.
Maximo 800 palavras. Inclua 1 frase "quotable" por secao (que alguem repostaria).
```

### 10. Sequencia de Posts de Autoridade

Para construir posicionamento como referencia em um tema.

```
Crie uma sequencia de 5 posts de LinkedIn para me posicionar como referencia em [tema/area].
Meu perfil: [cargo, empresa, experiencia].

Post 1: Opiniao controversa sobre [tema] (gerar debate)
Post 2: "X coisas que aprendi em [Y anos] fazendo [atividade]" (experiencia)
Post 3: Case de sucesso ou fracasso com licao (storytelling)
Post 4: Framework/metodo proprio para [resolver problema] (metodologia)
Post 5: Previsao/tendencia sobre o futuro de [area] (visao)

Para cada post: texto completo (150-200 palavras), melhor dia/horario para postar, e CTA.
```

## Multi-Plataforma

### 11. Conteudo Adaptado para Cada Rede

Para criar um unico conteudo e adaptar para todas as plataformas.

```
Tenho a seguinte mensagem/conteudo que quero publicar:
[Cole o conteudo ou descreva a ideia]

Adapte para cada plataforma mantendo a essencia:
1. Instagram Feed: legenda + descricao do visual + hashtags
2. Instagram Stories: sequencia de 5 stories com stickers
3. Instagram Reels: roteiro de 30 segundos
4. LinkedIn: post profissional (200 palavras)
5. Twitter/X: thread de 5 tweets
6. TikTok: roteiro de 60 segundos
7. WhatsApp Status: 3 imagens com texto

Para cada adaptacao, ajuste: tom, tamanho, formato e CTA de acordo com o comportamento do usuario em cada plataforma.
```

### 12. Calendario de Conteudo Multi-Plataforma

Para planejar 2 semanas de conteudo em todas as redes.

```
Crie um calendario de conteudo de 14 dias para [marca/profissional de nicho].
Plataformas: [Instagram, LinkedIn, Twitter/X, YouTube].

Para cada dia:
- Plataforma
- Formato (post, stories, reels, artigo, thread, video)
- Tema/titulo
- Objetivo (educar, engajar, vender, entreter)
- Briefing resumido (2 frases sobre o que abordar)

Regras:
- Maximo 2 publicacoes por dia (total, somando todas as plataformas)
- Alternar plataformas para nao sobrecarregar nenhuma
- 1 conteudo de venda por semana (maximo)
- Reaproveitar conteudo entre plataformas (indicar qual pode ser adaptado)
- Incluir datas relevantes para o nicho de [nicho]

Formato: tabela com colunas Dia, Plataforma, Formato, Tema, Objetivo.
```

## Engajamento e Comunidade

### 13. Respostas para Comentarios e DMs

Para manter o engajamento ativo sem perder tempo.

```
Crie templates de respostas para os tipos mais comuns de interacao em [plataforma] para [marca/perfil].

Tipos de comentario:
1. Elogio ao conteudo
2. Pergunta sobre o produto/servico
3. Pedido de preco
4. Critica construtiva
5. Comentario negativo/hater
6. Pedido de parceria/collab
7. Spam/off-topic
8. "Como faco para comecar?"

Para cada tipo:
- 2 opcoes de resposta (variar para nao parecer robotico)
- Maximo 30 palavras cada
- Tom: [amigavel/profissional/descontraido]
- Quando responder por DM ao inves de publicamente

Inclua: regra de tempo (responder em ate X horas) e quando escalar para o dono do perfil.
```

### 14. Prompts para Caixinha de Perguntas

Para gerar engajamento nos stories usando perguntas estrategicas.

```
Crie 15 perguntas para caixinha de perguntas do Instagram Stories para [perfil de nicho].

5 perguntas sobre o nicho:
(ex: "Qual sua maior dificuldade com [tema]?")

5 perguntas pessoais/bastidores:
(ex: "O que voces gostariam de saber sobre minha rotina?")

5 perguntas interativas:
(ex: "Isso ou aquilo?" / "Verdadeiro ou falso?")

Para cada pergunta, inclua:
- Texto da pergunta (maximo 60 caracteres)
- Tipo de sticker sugerido (pergunta, enquete, quiz, slider)
- Como usar as respostas (gerar conteudo, stories de resposta, etc.)
- Melhor dia/horario para postar

O objetivo e gerar respostas que possam ser transformadas em conteudo futuro.
```

### 15. Estrategia de Engajamento Diario

Para criar uma rotina de interacao que cresce o perfil organicamente.

```
Crie uma rotina diaria de engajamento para [plataforma] que leve no maximo 30 minutos por dia.

Perfil: [marca/profissional de nicho].
Seguidores atuais: [numero].
Meta: [ganhar seguidores/gerar leads/vender].

Divida em blocos:
Bloco 1 (10 min): Responder comentarios e DMs pendentes
Bloco 2 (10 min): Engajamento proativo (onde e como interagir)
Bloco 3 (10 min): Networking estrategico (com quem interagir)

Para cada bloco, detalhe:
- Acoes especificas (ex: comentar em 10 posts do hashtag X)
- O que evitar (ex: comentarios genericos de 1 palavra)
- Metricas para acompanhar (ex: taxa de resposta em DMs)

Inclua: checklist semanal de engajamento e como medir se a estrategia esta funcionando.
```

## Como Personalizar Estes Prompts

- **Defina sua brand voice**: antes de usar qualquer prompt, documente como sua marca fala. Inclua exemplos de posts que representam seu tom e peca a IA para seguir esse estilo.
- **Adapte ao seu publico**: o mesmo conteudo funciona diferente para publicos de 20 e 40 anos. Especifique faixa etaria, interesses e nivel de conhecimento.
- **Observe seus melhores posts**: analise quais posts tiveram mais engajamento nos ultimos 3 meses. Use esses como referencia nos prompts com "escreva no estilo deste post que performou bem: [cole]".
- **Teste e ajuste**: redes sociais sao experimentacao constante. Gere variantes, teste ambas e use os dados para refinar os prompts.
- **Mantenha autenticidade**: use a IA para estrutura e rascunho, mas adicione seu toque pessoal (experiencias reais, opinioes genuinas, humor proprio).
- **Planeje em lotes**: em vez de criar 1 post por vez, use os prompts de lote para produzir uma semana ou mes inteiro. Isso economiza tempo e garante consistencia.
