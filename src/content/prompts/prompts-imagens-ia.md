# 20 Prompts para Gerar Imagens com IA

Gerar imagens com IA e uma habilidade que depende diretamente da qualidade do prompt. A diferenca entre uma imagem amadora e uma profissional esta nos detalhes que voce inclui na descricao. Estes 20 prompts cobrem os casos mais comuns para negocios e criadores de conteudo.

## Estrutura de um Bom Prompt de Imagem

Antes dos prompts, entenda a formula basica que funciona na maioria das ferramentas:

```
[Sujeito] + [Acao/Composicao] + [Estilo artistico] + [Iluminacao] + [Camera/Angulo] + [Detalhes tecnicos]
```

Exemplo: "Uma mulher executiva em escritorio moderno, trabalhando em notebook, estilo fotografia corporativa, iluminacao natural pela janela, lente 50mm f/1.8, fundo desfocado"

## Logos e Identidade Visual

### 1. Logo Minimalista

Para gerar opcoes de logo para marcas novas ou rebrandings.

```
Logo minimalista para uma empresa chamada [nome] do segmento de [segmento]. Estilo flat design, maximo 2 cores ([cor 1] e [cor 2]), fundo branco. O icone deve remeter a [conceito]. Linhas limpas, sem gradientes, sem sombras. Adequado para uso em tamanho pequeno (favicon) e grande (outdoor). Vetor simplificado.
```

### 2. Logo com Icone e Tipografia

Para logos que combinam simbolo e texto.

```
Logo combinando icone e texto para [marca] do setor de [setor]. Icone: [descreva o simbolo desejado]. Tipografia: sans-serif moderna e bold. Cores: [paleta]. O icone deve funcionar sozinho e com o texto ao lado. Fundo transparente, estilo profissional e contemporaneo. Sem efeitos 3D.
```

## Mockups e Apresentacao de Produto

### 3. Mockup de Produto Digital

Para apresentar apps, sites ou produtos digitais de forma profissional.

```
Mockup de tela de aplicativo em um iPhone 15 Pro, segurando por uma mao feminina em ambiente de escritorio moderno. A tela mostra [descreva o que aparece na tela]. Iluminacao natural suave, fundo desfocado com plantas e mesa de madeira clara. Estilo fotografia de produto tech, alta resolucao, cores vibrantes mas nao saturadas.
```

### 4. Mockup de Produto Fisico

Para apresentar embalagens, livros ou produtos tangiveis.

```
Fotografia de produto de [descreva o produto] sobre fundo [cor/textura]. Composicao centralizada, iluminacao de estudio profissional com softbox, sombra suave. Angulo de [45 graus/frontal/top-down]. Acompanhado de [elementos de composicao: plantas, pedras, tecido, etc.]. Estilo editorial, alta resolucao, aspecto premium.
```

### 5. Mockup de Camiseta/Vestuario

Para marcas de moda ou merch.

```
Mockup de camiseta [cor] com estampa [descreva a estampa] vestida por [modelo: genero, etnia, idade]. Ambiente: [estudio fundo neutro / urbano / praia]. Iluminacao [natural/estudio]. O foco deve estar na estampa. Estilo lookbook de moda, fotografia limpa, sem filtros excessivos.
```

## Ilustracoes

### 6. Ilustracao para Blog/Artigo

Para criar imagens de destaque unicas para conteudos.

```
Ilustracao digital sobre o conceito de [tema do artigo]. Estilo: [flat illustration / isometrico / aquarela digital / line art]. Paleta de cores: [descreva 3-4 cores]. Composicao: [descreva os elementos visuais]. Sem texto na imagem. Proporcao 16:9, adequado para header de blog. Fundo [cor solida / gradiente suave / branco].
```

### 7. Ilustracao de Personagem (Avatar/Mascote)

Para criar identidade visual com personagem.

```
Personagem mascote para marca de [segmento]. O personagem e um(a) [animal/pessoa/objeto] com expressao [amigavel/confiante/divertida]. Vestindo [descreva roupas/acessorios que remetam ao segmento]. Estilo: [cartoon moderno / anime / pixar / flat]. 5 poses diferentes: acenando, pensando, comemorando, apontando para o lado, sentado trabalhando. Fundo transparente.
```

### 8. Infografico Visual

Para criar elementos visuais de infograficos.

```
Conjunto de 8 icones ilustrados para um infografico sobre [tema]. Estilo: [flat / outline / isometrico]. Tamanho uniforme, mesma paleta de cores ([liste cores]). Cada icone representa: [liste os 8 conceitos]. Fundo transparente, tracos consistentes, adequados para impressao e digital.
```

## Fotos de Produto

### 9. Foto de Comida/Gastronomia

Para restaurantes, delivery e marcas de alimentos.

```
Fotografia de [prato/produto alimenticio] em prato [tipo/cor] sobre mesa de [material]. Food styling profissional. Iluminacao natural vindo da esquerda, sombras suaves. Guarnicao: [ervas, molho, acompanhamentos]. Angulo: [45 graus / top-down / close-up]. Estilo editorial de revista gastronomica. Cores vibrantes e naturais. Resolucao alta.
```

### 10. Foto de Produto para E-commerce

Para catalogos e fichas de produto de lojas online.

```
Fotografia de produto para e-commerce: [descreva o produto em detalhe]. Fundo branco puro (#FFFFFF). Iluminacao de estudio uniforme sem sombras duras. 3 angulos: frontal, lateral 45 graus, detalhe de [caracteristica]. Produto centralizado com margem de 15% nas bordas. Estilo Amazon/Mercado Livre, profissional e limpo.
```

## Banners e Materiais Graficos

### 11. Banner para Redes Sociais

Para criar capas e banners que se destacam no feed.

```
Banner para [Instagram/Facebook/LinkedIn/YouTube] sobre [tema/promocao]. Dimensoes: [especifique - ex: 1080x1080, 1200x628, 1584x396]. Elementos: [descreva o que deve aparecer]. Espaco reservado para texto na [posicao]. Paleta: [cores da marca]. Estilo: [moderno/minimalista/vibrante/corporativo]. Sem texto na imagem (sera adicionado depois).
```

### 12. Banner de Promocao/Oferta

Para campanhas de venda e promocoes.

```
Banner promocional para [Black Friday / lancamento / liquidacao] de [produto/segmento]. Fundo [cor forte que transmite urgencia]. Espaco para: headline grande no topo, preco/desconto centralizado, CTA no rodape. Elementos graficos: [confetes, raios, formas geometricas]. Estilo: energetico e chamativo mas nao poluido. Paleta: [cores]. Proporcao: [quadrado / retangular / stories].
```

### 13. Background para Apresentacao

Para criar fundos profissionais para slides.

```
Fundo para slide de apresentacao corporativa sobre [tema]. Estilo: [clean/tecnologico/criativo/formal]. Cor predominante: [cor da marca]. Elementos sutis: [formas geometricas abstratas / gradiente suave / padrao minimalista / textura]. Espaco livre para texto no [centro/esquerda/direita]. Sem elementos que distraiam. Proporcao 16:9. Profissional e sofisticado.
```

## Thumbnails e Capas

### 14. Thumbnail para YouTube

Para criar thumbnails que geram cliques.

```
Thumbnail para video de YouTube sobre "[titulo do video]". Proporcao 16:9 (1280x720). Composicao: [pessoa com expressao exagerada na direita, texto grande na esquerda] ou [antes e depois] ou [3 elementos em sequencia]. Cores: alto contraste, [amarelo/vermelho/azul] dominante. Estilo: energetico, impactante, visivel em tela pequena. Espaco para texto bold grande.
```

### 15. Capa de Ebook/Lead Magnet

Para criar capas profissionais para materiais digitais.

```
Capa de ebook digital sobre "[titulo]". Estilo: [moderno/corporativo/criativo]. Proporcao: retrato (A4 ou similar). Elementos: [ilustracao abstrata / foto de fundo / padrao grafico]. Espaco para: titulo grande, subtitulo, nome do autor, logo. Paleta: [cores]. O design deve transmitir [credibilidade/inovacao/praticidade]. Sem fotos stock genericas.
```

### 16. Capa de Podcast

Para criar a identidade visual do podcast.

```
Capa de podcast chamado "[nome do podcast]" sobre [tema]. Formato: quadrado (3000x3000px). O design deve funcionar em tamanho de icone (pequeno no app). Elementos: [descreva]. Tipografia: bold e legivel em tamanho pequeno. Cores: [paleta com alto contraste]. Estilo: [minimalista/ilustrado/fotografico]. Nao incluir texto alem do nome (sera adicionado separadamente).
```

## Fotos Conceituais

### 17. Foto para Storytelling de Marca

Para transmitir valores e cultura da empresa.

```
Fotografia conceitual que transmite [valor: inovacao/confianca/colaboracao/liberdade/crescimento]. Cena: [descreva a situacao]. Pessoas: [descreva perfis, diversidade, vestuario]. Ambiente: [escritorio/natureza/urbano/estudio]. Iluminacao: [golden hour/natural/cinematografica]. Estilo: [editorial/lifestyle/documental]. Cor predominante: [cor]. Sensacao geral: [descreva a emocao desejada].
```

### 18. Foto de Equipe/Cultura

Para paginas "Sobre nos" e materiais de employer branding.

```
Fotografia de equipe diversa de [5-8 pessoas] em [escritorio moderno / coworking / area externa]. Expressoes: naturais e genuinas (nao posadas). Atividade: [reuniao informal / brainstorm com post-its / celebrando / trabalhando junto]. Vestuario: smart casual. Iluminacao natural. Estilo: corporativo humanizado, nao stock generico. Foco na interacao entre as pessoas.
```

## Como Personalizar Estes Prompts

- **Especifique a ferramenta**: cada IA de imagem tem particularidades. Midjourney funciona bem com estilos artisticos e parametros como --ar 16:9 e --v 6. DALL-E prefere descricoes mais literais. Stable Diffusion aceita negative prompts.
- **Adicione parametros tecnicos**: para Midjourney, use --ar (aspect ratio), --s (stylize), --c (chaos), --q (quality). Para DALL-E e outras, especifique resolucao e formato.
- **Use referencias de estilo**: "no estilo de [artista/fotografo/epoca/revista]" ajuda muito. Exemplos: "estilo Wes Anderson", "fotografia editorial Vogue", "ilustracao Notion".
- **Itere removendo elementos**: se a imagem tiver algo indesejado, adicione "sem [elemento]" ou use negative prompts.
- **Gere variantes**: sempre gere 4 ou mais opcoes e selecione a melhor. Faca pequenos ajustes no prompt para explorar direcoes diferentes.
- **Combine com edicao**: a imagem da IA e o ponto de partida. Use Photoshop, Canva ou Figma para adicionar texto, ajustar cores e finalizar para seu uso.
- **Mantenha consistencia**: para uma serie de imagens (posts, paginas), mantenha a mesma descricao de estilo, iluminacao e paleta. Salve um "prompt base" e so mude o sujeito.
