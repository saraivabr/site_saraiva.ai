# 15 Prompts de IA para Vendas e Copywriting

Se voce trabalha com vendas ou marketing, sabe que escrever textos persuasivos consome tempo. A boa noticia e que a IA pode acelerar drasticamente esse processo. Abaixo estao 15 prompts prontos para copiar, colar e adaptar ao seu negocio.

## Headlines e Titulos

### 1. Gerador de Headlines com Gatilhos Mentais

Use quando precisar de opcoes rapidas de titulos para anuncios, emails ou paginas de venda.

```
Crie 10 headlines para vender [produto/servico] para [publico-alvo].
Use os seguintes gatilhos mentais: urgencia, escassez, prova social e curiosidade.
Formato: uma headline por linha, maximo de 12 palavras cada.
Tom: direto e persuasivo.
```

### 2. Titulos para Testes A/B

Ideal para quem roda campanhas de anuncios e precisa de variantes para testar.

```
Gere 5 variantes de titulo para um anuncio do [produto].
Publico: [descreva o publico].
Cada variante deve usar uma abordagem diferente: beneficio direto, pergunta, dado numerico, depoimento e comparacao.
Limite: 8 palavras por titulo.
```

## Emails de Venda

### 3. Email de Lancamento de Produto

Para quando voce precisa anunciar algo novo para sua base de contatos.

```
Escreva um email de lancamento para [produto/servico].
Estrutura: assunto chamativo (maximo 50 caracteres), abertura com gancho emocional, 3 beneficios principais em bullet points, prova social (mencione [numero] clientes ou [resultado]), CTA claro para [acao desejada].
Tom: entusiasmado mas profissional. Maximo 200 palavras no corpo.
```

### 4. Sequencia de Emails de Nurturing

Use para criar uma sequencia automatizada que aquece leads ao longo de dias.

```
Crie uma sequencia de 5 emails para nutrir leads interessados em [produto/servico].
Email 1: Boas-vindas + entrega de conteudo gratuito
Email 2: Historia de transformacao de um cliente
Email 3: Conteudo educativo sobre [tema relacionado]
Email 4: Oferta com urgencia
Email 5: Ultimo lembrete + bonus exclusivo
Para cada email, inclua: assunto, corpo (maximo 150 palavras) e CTA.
```

### 5. Email de Recuperacao de Carrinho

Para e-commerces que precisam recuperar vendas abandonadas.

```
Escreva 3 versoes de email de recuperacao de carrinho abandonado para uma loja de [segmento].
Versao 1: lembrete gentil (enviar 1h depois)
Versao 2: beneficio adicional (enviar 24h depois)
Versao 3: urgencia com prazo (enviar 48h depois)
Inclua assunto e corpo para cada. Tom: amigavel, sem ser insistente.
```

## Scripts de Venda

### 6. Script para Ligacao Comercial

Para vendedores que fazem prospecao ativa por telefone.

```
Crie um script de ligacao comercial para vender [produto/servico] para [cargo/perfil do decisor].
Estrutura:
- Abertura (15 segundos): apresentacao + motivo da ligacao
- Qualificacao (3 perguntas para entender a dor)
- Apresentacao da solucao (maximo 60 segundos)
- Tratamento de objecoes (3 objecoes mais comuns e respostas)
- Fechamento com proximo passo claro
Tom: consultivo, nao agressivo.
```

### 7. Script para Video de Vendas (VSL)

Para quem precisa de um roteiro de video que converte.

```
Escreva o roteiro de um video de vendas de 3 minutos para [produto/servico].
Estrutura:
- Gancho (primeiros 10 segundos): pergunta ou dado impactante
- Problema: descreva a dor do [publico-alvo]
- Agitacao: amplifique as consequencias de nao resolver
- Solucao: apresente [produto] como a resposta
- Prova: mencione [resultados/depoimentos]
- Oferta: preco, bonus, garantia
- CTA: instrucao clara do proximo passo
```

## Landing Pages

### 8. Estrutura Completa de Landing Page

Para criar paginas de conversao do zero.

```
Crie o conteudo completo de uma landing page para [produto/servico].
Publico-alvo: [descreva].
Inclua:
- Headline principal + sub-headline
- Secao de problema (3 dores do publico)
- Secao de solucao (como o produto resolve)
- 5 beneficios com icone + titulo + descricao curta
- 3 depoimentos ficticios realistas
- FAQ com 5 perguntas
- CTA principal e secundario
Tom: profissional e persuasivo. Foco em conversao.
```

### 9. Bullet Points de Beneficios

Para refinar a secao de beneficios de qualquer pagina.

```
Transforme estas caracteristicas do [produto] em beneficios orientados ao cliente:
[Liste 5-7 caracteristicas tecnicas]
Para cada uma, escreva: um titulo de beneficio (maximo 5 palavras) e uma frase explicando o impacto real na vida do cliente (maximo 20 palavras).
Use a formula: "Voce consegue [resultado] sem [obstaculo]".
```

## Tratamento de Objecoes

### 10. Respostas para Objecoes Comuns

Para preparar a equipe comercial com respostas prontas.

```
Liste as 10 objecoes mais comuns na venda de [produto/servico no segmento X] e crie uma resposta persuasiva para cada.
Formato para cada objecao:
- Objecao: [frase do cliente]
- Resposta: [validacao da preocupacao + reframe + prova ou exemplo]
Maximo 3 frases por resposta. Tom: empatico e seguro.
```

### 11. Comparativo com Concorrente

Para quando o prospect menciona um concorrente.

```
Crie um comparativo entre [meu produto] e [concorrente] focado em valor, nao em ataque.
Estrutura: tabela com 5 criterios relevantes para o [publico-alvo].
Apos a tabela, escreva um paragrafo de 50 palavras explicando por que [meu produto] e a melhor escolha para [perfil especifico de cliente].
```

## Propostas e Apresentacoes

### 12. Proposta Comercial por Email

Para enviar apos uma reuniao de discovery.

```
Escreva uma proposta comercial por email para [empresa/cliente] apos uma reuniao onde identificamos os seguintes problemas: [liste 2-3 problemas].
Estrutura: recapitulacao dos problemas, solucao proposta com [produto/servico], escopo do trabalho em 3-5 itens, investimento, proximo passo.
Tom: profissional e personalizado. Maximo 300 palavras.
```

### 13. Pitch de Elevador

Para apresentacoes rapidas do seu negocio.

```
Crie 3 versoes de pitch de elevador para [empresa/produto]:
- Versao 30 segundos: para networking casual
- Versao 60 segundos: para reuniao com investidor
- Versao 2 minutos: para apresentacao formal
Inclua: problema que resolve, como resolve, diferencial e resultado mensuravel.
```

## Follow-up e Pos-Venda

### 14. Mensagens de Follow-up

Para manter o contato ativo sem ser inconveniente.

```
Crie 5 mensagens de follow-up para enviar a um lead que nao respondeu apos receber uma proposta de [produto/servico].
Mensagem 1: 2 dias depois (check-in leve)
Mensagem 2: 5 dias depois (agregar valor com conteudo)
Mensagem 3: 10 dias depois (caso de sucesso relevante)
Mensagem 4: 15 dias depois (nova abordagem/angulo)
Mensagem 5: 30 dias depois (ultimo contato gentil)
Canal: [email/WhatsApp]. Maximo 50 palavras cada.
```

### 15. Solicitacao de Depoimento

Para coletar provas sociais de clientes satisfeitos.

```
Escreva um pedido de depoimento para enviar a um cliente que comprou [produto/servico] ha [tempo].
Inclua 3 perguntas guia para facilitar a resposta:
1. Qual era o problema antes?
2. O que mudou depois de usar [produto]?
3. Voce recomendaria? Por que?
Tom: caloroso e agradecido. Facilite ao maximo a resposta (opcao de responder por audio).
```

## Dicas de Uso

- **Personalize os campos entre colchetes**: quanto mais especifico voce for ao preencher [produto], [publico-alvo] e [segmento], melhores serao os resultados.
- **Itere sobre as respostas**: o primeiro resultado raramente e perfeito. Peca para a IA ajustar o tom, encurtar ou expandir trechos especificos.
- **Combine prompts**: use o prompt de headlines junto com o de landing page para criar campanhas completas.
- **Teste variantes**: gere multiplas versoes e teste com seu publico real antes de escalar.
- **Adapte ao seu tom de voz**: adicione no prompt algo como "use o tom de voz de [marca/referencia]" para manter consistencia.
- **Salve os melhores resultados**: crie uma biblioteca interna dos textos que mais converteram para usar como referencia em prompts futuros.
